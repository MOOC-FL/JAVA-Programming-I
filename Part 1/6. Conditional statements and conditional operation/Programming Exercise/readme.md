#### Exercise Details
##### Programming exercise: Speeding Ticket
*Write a program that asks the user for an integer and prints the string "Speeding ticket!" if the input is greater than 120.*
##### Programming exercise: Check Your Indentation
*Fix the indentation in a given program to ensure proper code structure and readability.*
##### Programming exercise: Orwell
*Write a program that prompts the user for an integer and prints "Orwell" if the number is exactly 1984.*
##### Programming exercise: Ancient
*Write a program that asks the user for a year and prints "Ancient history!" if the entered value is less than 2015.*
##### Programming exercise: Positivity
*Write a program that asks the user for a number and prints the number if it is positive.*
##### Programming exercise: Adulthood
*Write a program that asks the user for their age and prints "You are an adult" if the age is 18 or more.*
##### Programming exercise: Larger Than or Equal To
*Write a program that asks the user for two numbers and prints the larger one. If they are equal, it should print "The numbers are equal!".*
##### Programming exercise: Grades and Points
*Write a program that converts point values to grades according to specific criteria (e.g., 0-29 points = failed, 30-34 points = 1, etc.).*
##### Programming exercise: Odd or even
*Write a program that asks the user for an integer and determines whether it is odd or even.*
##### Programming exercise: Password
*Write a program that asks the user for a password. If the password is correct ("Caput Draconis"), it should print "Welcome!". Otherwise, it should print "Off with you!".*
##### Programming exercise: Same
*Write a program that asks the user for two strings. If the strings are equal, it should print "Same". Otherwise, it should print "Different".*
##### Programming exercise: Checking the age
*Write a program that asks for the user's age and checks if it is within a reasonable range (e.g., 0-120 years).*

##### Programming exercise: Leap year
*Write a program that determines whether a given year is a leap year according to the standard rules.*

##### Programming exercise: Gift tax
*Write a program that calculates the gift tax for a given gift value according to specific tax brackets and rules.*
